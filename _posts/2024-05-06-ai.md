### AI模型prompt的书写
> 当与AI聊天时，prompt是指您输入的问题、主题或起始语句，用于引导AI生成回答或继续对话。Prompt通常是一个简短的句子或问题，目的是为了指导AI理解您想要讨论的话题或回答您的问题。在与AI聊天时，一个好的prompt可以帮助AI更好地理解您的意图并生成相关、有意义的回复。
> 写一个好的、精准的prompt有几个关键要点：
1. **清晰明了的问题或任务**: 确保你的prompt清楚表达出你想要的信息或结果，避免模糊或含糊不清的描述。

2. **具体的指导**: 提供必要的上下文和具体的指导，包括所需的输入、输出、格式要求等。让读者明白需要做什么。

3. **简洁明了**: 避免冗长或复杂的句子，尽量用简洁的语言描述问题或任务。

4. **明确的期望**: 说明预期的结果或行为，确保读者理解他们应该如何回应或采取行动。
     例如，一个好的prompt可以是：“请编写一个函数，接受一个字符串作为输入，并返回该字符串的反转版本。”这个prompt清晰明了，指明了输入和输出的要求，也提供了明确的任务。

     


## 市面上AI工具的使用
### 电脑开启代理——以Clash为例
1. 打开桌面上的Clash For Windows软件：
   ![Clash](https://img2.imgtp.com/2024/05/06/eLC6ixC8.png)

   

2. 开启下方的`系统代理`选项（程序主开关）
   注意：在使用完AI或者翻墙需求后，回到软件<font color="red">先关闭系统代理选项</font>
   ![Open](https://img2.imgtp.com/2024/05/06/TR15cfud.png)

   

3. 点击左方的`代理`选项，进入节点选择页面，<font color="red">先点击右上角的WIFI样式按钮</font>进行延迟测试
   ![Speed Test](https://img2.imgtp.com/2024/05/06/PP588lg2.png)

   

4. 在<font color="red">`日本`，`新加坡`，`美国`</font>的节点中，选择右方存在数字且数字最小的一个节点
   ![Node Select](https://img2.imgtp.com/2024/05/06/C6HyfrwY.png)
   注意：如果所有节点都不存在数字的情况下，和我说！！

   
### 手机端代理开启
1. 打开手机上的Clash软件
   <img src="https://img2.imgtp.com/2024/05/06/BYu8yH3a.jpg" alt="手机Clash"/>

   

2. 点击上方的启动按钮启动，当手机状态栏上显示VPN时即为成功
   ![开启](https://img2.imgtp.com/2024/05/06/8jaDjQeE.jpg)

   

3. 点击中间的代理按钮，进入节点选择，<font color="red">先点击右下角的闪电按钮</font>进行延迟测试
   ![延迟测试](https://img2.imgtp.com/2024/05/06/I2uhVYnO.jpg)

   

4. 同样在<font color="red">`日本`，`新加坡`，`美国`</font>的节点中，选择右方存在数字且数字最小的一个节点
   ![日本节点](https://img2.imgtp.com/2024/05/06/MPo2JQf0.jpg)

   
### ChatGPT使用
1. 登录地址：[ChatGPT登录地址](https://chat.openai.com)
   注意：如果长时间加载不出来，请重新回到Clash，然后重新进行<font color="red">延迟测试</font>（电脑端的代理—>WIFI按钮，手机端的代理—>闪电按钮），延迟测试后重新选择一个能正常使用的节点。

   

2. 电脑端登录，网页打开后点击左下角的`Log in`，输入账号（邮箱号）
    ![gpt登录](https://img2.imgtp.com/2024/05/06/A48dWyo4.jpg)

  

3. 输入邮箱号后，点击`继续`按钮，进入下一步输入密码，如果不需要GPT4的功能可登录个人账号，需要GPT4找我要！
   注意：登录过程中可能会跳出CloudFlare认证页面，属正常现象，点击旁边的按钮确认即可。
   ![输入邮箱](https://img2.imgtp.com/2024/05/06/XO5bG5M1.jpg)

   

4. 输入对应账号的密码后，点击`继续`按钮
    ![确认登录](https://img2.imgtp.com/2024/05/06/RmzcxxSQ.jpg)

  

5. 登录成功后进入主界面，其中页面分为两个部分，左边部分为当前账户的所有聊天的<font color="red">历史记录</font>，右半部分为当前聊天的交互窗口（如果是新聊天则什么都没有）
    ![页面介绍左](https://img2.imgtp.com/2024/05/06/LcN0vuj2.jpg)
    ![页面介绍右](https://img2.imgtp.com/2024/05/06/AIrm4k8N.jpg)

  

6. 手机端登录以及使用，同样点击对应的链接 [ChatGPT登录地址](https://chat.openai.com) ，或者在输入框中输入chat，然后根据历史记录查找第一个，点击页面<font color="red">左上角的详细按钮</font>后，即可看见登录按钮`Log In`以及<font color="red">历史记录部分</font>

  ![手机端登录](https://img2.imgtp.com/2024/05/06/Bpt3BqUG.jpg)

  ![Log In](https://img2.imgtp.com/2024/05/06/3lKBNUxX.jpg)

  

7. 登录成功后显示聊天主页面，如果要查看历史记录等请点击左上角详细按钮（两条横杠）

   ![手机页面](https://img2.imgtp.com/2024/05/06/7VK7zl74.jpg)

   

8. GPT聊天回答下方对应按钮的作用

   

   注意：GPT的每一次聊天都会存在<font color="red">“聊天上下文机制”</font>，意思为GPT会记住你每一次提出的要求等内容，比如我在聊天一开始和GPT说

   

   ```
   You: 
   	请你介绍一下什么是GPT，所有的回答都使用中文
   ChatGPT:
   	GPT是“生成预训练模型”的缩写，是一种自然语言处理模型。它是由OpenAI开发的，旨在理解和生成人类语言。GPT模型使用了深度学习技术，特别是变压器（Transformer）架构，这使得它在处理文本任务方面非常出色。GPT模型通过预先训练大规模文本数据，学习了语言的结构和语境，从而能够生成连贯、合乎逻辑的文本，也能够执行各种文本相关任务，如问答、翻译、摘要等。
   ```

   

   这段对话中我指定了回答使用中文，那GPT会在<font color="red">当前聊天</font>中全程使用中文，但如果创建一个新聊天后，GPT则会忘记我的要求。

   

   <font color="red">历史记录功能不仅能恢复当时的聊天，还能恢复当时聊天的上下文</font>（包含你当时的要求等）

   

   ![回答作用](https://img2.imgtp.com/2024/05/06/vFpQc1bz.png)

#### GPT3.5和GPT4的区别

|  | GPT3.5 | GPT4 |
| :-: | :-: | --- |
| 联网功能 | × | √ |
| 附件功能 | × | **各类文档以及图片** |
| 知识库 | 截至21年9月 | **有联网功能实时获取** |
| 插件功能 | × | **存在插件市场** |
| 聊天次数限制 | **无限制** | 3小时/40次回复 |
| 智能性 | 一般 | **GPT4能处理更复杂的任务** |



#### GPT4插件市场的使用

当登录GPT4的账号以后，点击左上角的`Explore GPTs`按钮，即可进入插件市场

![插件市场](https://img2.imgtp.com/2024/05/06/isj0j39c.png)



插件市场能让GPT4更加智能以及拓展更多的功能，下滑到`Trending`部分即可看见常用插件，常用的几个插件有：

- Image Generator

  一个专门用于生成和完善的GPT4<font color="red">图像生成器</font>（AI画图）。

  ![图片生成](https://img2.imgtp.com/2024/05/06/DaWGbG9f.png)

- Write For Me

  一个专门用于书写内容的插件，注重质量、相关性和精确字数。

  ![Write For Me](https://img2.imgtp.com/2024/05/06/R7lGYHWG.png)

- Consensus

  人工智能研究助手，搜索来自 Consensus 的 2 亿篇学术论文，获得基于科学的答案，并起草带有准确引文的内容。

  ![Consensus](https://img2.imgtp.com/2024/05/06/SLonrVMm.png)

- Scholar GPT

  利用 200M+ 资源和内置的批判性阅读技能，提高研究水平。轻松访问 Google Scholar、PubMed、JSTOR、Arxiv （这些都是国外学术平台）等。

  ![学术GPT](https://img2.imgtp.com/2024/05/06/J29umvqV.png)

- Canva（可画）

  轻松设计任何内容：演示文稿、徽标、社交媒体帖子等。

  注意：Canva 插件的功能实际就是跟 ChatGPT 说你要制作怎么样的图像作品，Canva 会向你推荐符合的模板，并给你模板链接，点击进去可以直接编辑，<font color="red">并不是直接生成</font>，只是节省了挑选模板的时间。

  ![Canva](https://img2.imgtp.com/2024/05/06/9uoOzl6y.png)

- Logo Creator

  生成专业的徽标设计和应用程序图标。

  ![Logo Creator](https://img2.imgtp.com/2024/05/06/cJ5SebO5.png)



### Copilot的设置

> Copilot是微软使用**GPT4**作为核心内置在**Bing**搜索引擎中的AI助手

Copilot具有网络搜索，图片上传以及识别等功能，<font color="red">不存在聊天次数限制</font>，但是因为内置在Bing，我们需要<font color="red">一个微软账号</font>，并且<font color="red">修改地区到美国</font>

1. 在浏览器中（推荐使用Windows自带Edge浏览器）打开[Bing](https://www.bing.com)后点击右上角的登录，登录微软账号，选择个人账户登录。

   ![1714993628073.png](https://img2.imgtp.com/2024/05/06/sY0lDldH.png)

2. 在账户成功登录后，随便搜索内容或者查看页面上是否存在`Copilot`相关内容，如果存在则可直接使用，不存在请先<font color="red">切换地区</font>

   ![Copoilot可以使用](https://img2.imgtp.com/2024/05/06/1YkrrAXV.png)

3. 如不存在上方的内容，请先切换地区到美国，点击右上方的三条横杠，依次点击`设置——>国家/地区`

   ![国家地区](https://img2.imgtp.com/2024/05/06/iDAJw9l5.png)

   将地区修改为美国或日本，修改完后即会出现Copilot相关内容

   ![地区修改](https://img2.imgtp.com/2024/05/06/mVTveuVs.png)



### Copilot的使用

Copilot内置在Bing中，而Bing是搜索引擎，所以当我们<font color="red">搜索任何内容</font>都可以跳转到Copilot界面，默认会对你搜索的内容进行回答。

![Copilot](https://img2.imgtp.com/2024/05/06/YN5EpTg9.png)

上面的按钮随便一个都可以跳转到于Copilot聊天中，在进入Copilot页面后即可当作正常AI进行聊天以及回复。

注意：`Copilot`页面<font color="red">不能切屏</font>，切屏后会暂停继续回答，请等待完全回复以后再进行下一步操作。

![盐酥鸡](https://img2.imgtp.com/2024/05/06/uISuxPNN.png)

Copilot有每一次聊天的回复次数限制，未登录账户为5次，登录账户为30次，当次数用完时点击新主题按钮即可开启下一次聊天。

![聊天次数](https://img2.imgtp.com/2024/05/06/ywj8oa5R.png)



Copilot的聊天页面右方可以选择不同的模型，现有：

![模型选择](https://img2.imgtp.com/2024/05/06/O6nPBpso.png)

  ![图片生成](https://img2.imgtp.com/2024/05/06/2n08ptQh.png)

- Vacation planner：旅行计划

  ![Vacation planner](https://img2.imgtp.com/2024/05/06/zNH4VLia.png)

- Cooking assistant：做饭帮手

  ![Cooking assistant](https://img2.imgtp.com/2024/05/06/mpk5RtyT.png)
